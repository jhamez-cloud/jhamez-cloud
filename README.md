## About Me

I'm James — a software developer who enjoys working across the stack, from web apps to embedded systems. I'm equally comfortable shipping a full-stack MERN application, building a desktop tool with Windows Forms, or wiring up a microcontroller project with Arduino.

- Currently building full-stack apps with **Next.js**, **React**, and **Node.js**
- Also working with **Django**, **C++**, and **Windows Forms** for desktop & backend systems
- Tinkering with **Arduino** for embedded/hardware projects
- Open to collaborating on web, backend, and IoT/embedded projects
- Reach me at **jameskekeli12@gmail.com**
